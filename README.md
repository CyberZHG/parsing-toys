# Parsing Toys

https://cyberzhg.github.io/parsing-toys/

Toy tools for visualizing parsing algorithms.

## Context-Free Grammar

- [Context-Free Grammar Reformat](https://cyberzhg.github.io/parsing-toys/tools/cfg_reformat.html) - Parse and reformat context-free grammars.
- [Left Factoring](https://cyberzhg.github.io/parsing-toys/tools/cfg_left_factoring.html) - Find and group the longest common prefixes in a context-free grammar.
- [Left Recursion Elimination](https://cyberzhg.github.io/parsing-toys/tools/cfg_left_recursion.html) - Eliminate left recursions in a context-free grammar.
- [First & Follow Sets](https://cyberzhg.github.io/parsing-toys/tools/cfg_first_and_follow.html) - Compute the first & follow sets in a context-free grammar.
- [Chomsky Normal Form](https://cyberzhg.github.io/parsing-toys/tools/cfg_cnf.html) - Convert a context-free grammar to Chomsky Normal Form.
- [CYK Parser](https://cyberzhg.github.io/parsing-toys/tools/cfg_cyk.html) - Parse strings using the CYK (Cocke-Younger-Kasami) algorithm.
- [LL(1) Parsing](https://cyberzhg.github.io/parsing-toys/tools/cfg_lr0.html) - Generate LL(1) predictive parsing table and simulate parsing.
- [LR(0) Parsing](https://cyberzhg.github.io/parsing-toys/tools/cfg_lr0.html) - Generate LR(0) automaton, ACTION/GOTO table, and simulate parsing.
- [SLR(1) Parsing](https://cyberzhg.github.io/parsing-toys/tools/cfg_slr1.html) - Generate SLR(1) automaton, ACTION/GOTO table, and simulate parsing.
- [LR(1) Parsing](https://cyberzhg.github.io/parsing-toys/tools/cfg_lr0.html) - Generate LR(1) automaton, ACTION/GOTO table, and simulate parsing.
- [LALR(1) Parsing](https://cyberzhg.github.io/parsing-toys/tools/cfg_slr1.html) - Generate LALR(1) automaton, ACTION/GOTO table, and simulate parsing.

## Regular Expression

- [Regex to NFA/DFA/Min-DFA](https://cyberzhg.github.io/parsing-toys/tools/cfg_slr1.html) - Convert regular expression to NFA, DFA, and minimized DFA.
